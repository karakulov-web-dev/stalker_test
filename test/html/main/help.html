Настройка

Для просмотра необходимо установить VLC-плагин. Скачать VLC можно здесь.

На данный момент поддерживаются следующие браузеры: Mozilla Firefox, Opera.

Google Chrome

C 1 сентября 2015 Google отключил поддержку плагина VLC в своём браузере.

Чтобы и дальше использовать tv.rikt.ru, выберите другой браузер.

Использование приложения

При первом запуске приложения необходимо создать аккаунт нажав кнопку Создать аккаунт.

При деактивации вкладки браузера приложения КЛИК-ТВ, воспроизведения видео автоматически приостанавливается. Для возобновления воспроизведения нажмите кнопку .
